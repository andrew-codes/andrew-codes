## Overview

Today I'd like to share some of my developer workbench and the software I regularly use. I have broken them into a few categories:

<dl>
  <dt>Essential Software</dt>
  <dd>
    <p>Software I expect to always be installed on my dev machine.</p>
  </dd>
  <dt>Implementation Software</dt>
  <dd>
    Non-essential software that may be on my local development machine or
    installed in a dev container.
  </dd>
  <dt>VS Code Extensions</dt>
  <dd>
    <p>Extensions I use in VS Code.</p>
  </dd>
</dl>

> Remember, my primary development machine is running OSX.
